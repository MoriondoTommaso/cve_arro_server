# Data Directory

This directory contains **pre-computed CVE embeddings** and analysis results.
The raw embedding files are **not committed** to the repository (see `.gitignore`).

## Structure

```
data/
├── cve_embeddings_demo/
│   ├── embs_99_to_14.npy   # Embeddings for CVE 1999–2014  (~384-dim float32)
│   └── embs_15_to_2025.npy # Embeddings for CVE 2015–2025  (~384-dim float32)
├── cve_zarr/
│   ├── cve_99_2014.zarr/   # Zarr store for Period A
│   └── cve_15_2025.zarr/   # Zarr store for Period B
└── results/
    ├── spectral_drift_overview.png   # generated by notebooks/cve_spectral_drift.ipynb
    ├── yearly_drift_timeseries.png   # generated by notebooks/cve_drift_monitoring.ipynb
    ├── yearly_spectral_drift.csv     # generated by notebooks/cve_drift_monitoring.ipynb
    └── drift_metrics.json            # generated by scripts/compute_drift.py
```

## How to Regenerate Embeddings

The `.npy` files are produced by `notebooks/cve.ipynb` from the full parquet corpus:

```python
# cve.ipynb — final cells
df = pd.read_parquet('data/data/cve_embs/cve_corpus.parquet')
np.save('data/cve_embeddings_demo/embs_99_to_14.npy',
        df[df.year <= 2014]['embedding'].to_numpy())
np.save('data/cve_embeddings_demo/embs_15_to_2025.npy',
        df[df.year > 2014]['embedding'].to_numpy())
```

## Embedding Model

Embeddings were produced by a sentence-transformer model fine-tuned on NVD descriptions.
Each vector has dimension **384** (float32). One row = one CVE entry.

## Quick Start

```bash
# Run the drift analysis (no notebooks required)
uv run python scripts/compute_drift.py

# With custom parameters
uv run python scripts/compute_drift.py --n-sample 3000 --k 15 --n-eigs 150
```
